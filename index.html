About

GET STARTED WITH WEB SERVICES

Web Platforms hosts content and custom-developed applications for campus users. Users can host content in Drupal instances, or create applications in any supported language. Unlike hosting from outside vendors, our service provides ready access to campus resources and can be paid for with a shortcode. Your environment will be kept up-to-date with the latest security updates, and you’ll be provided with a non-production environment to verify application functionality prior to updating your production environment.

Services

EXPLORE YOUR OPTIONS

Cloud

Hosting is provided in the cloud, except when circumstances--such as the presence of sensitive data--dictates hosting occur on-premise.

Cloud hosted instances can access on-campus networks and services like Shibboleth, MCommunity, Email, Active Directory, Data Warehouse and the Enterprise Service Bus.

Customize

Users can choose custom URLs.

Choose from the following environments: Drupal 7, Python 2 and 3, Java, Ruby and NodeJS

Security

Sites and applications are scanned for security vulnerabilities, and receive regular security updates.

Certificates are provided free, enabling secure traffic to your site.

The supported environments are patched for security.

Develop

Developers can use a continuous integration (CI) pipeline to update test instances upon committing changes to their source code repository.

A non-production environment is provided to verify application functionality prior to updating your production environment.

Scale

You can choose to have your application to automatically scale up to meet user traffic demands.

Support

Application logs are available for 3 months.
